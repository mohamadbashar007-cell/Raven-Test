# Raven SPM Intelligence Test

تطبيق اختبار رافن للمصفوفات المتتابعة تحت اسم مؤسسة الإمام أحمد بن حنبل.

## التشغيل المحلي

```bash
npm install
npm run dev
```

## البناء للنشر

```bash
npm run build
```

ينتج البناء داخل مجلد `dist`. إعداد Vite يستخدم `base: "./"` لذلك تعمل الصور والملفات على GitHub Pages حتى لو كان الموقع منشوراً داخل مسار الريبو.
